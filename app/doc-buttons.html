按钮

轻松创建多种样式按钮，改变一个类即可调整外观，颜色，大小

可作为按钮使用的标签或元素

默认样式
Link 
<!-- 可使用的标签 -->
<a class="v-button" href="#">Link</a>
<button class="v-button" type="submit">Button</button>
<input class="v-button" type="button" value="Input">
<input class="v-button" type="submit" value="Submit">

使用下面列出的类可以快速创建一个带有预定义样式的按钮

外观： 默认 .line .radius .circle .link .min-width
颜色： 默认 .primary .secondary .demitint
图标： 增加 <i class="icon-**"> 


<!-- 预定义样式 -->
<button type="button" class="v-button primary">Button</button>
<button type="button" class="v-button primary line">Button</button>
<button type="button" class="v-button primary radius">Button</button>
<button type="button" class="v-button primary circle">C</button>
<button type="button" class="v-button link">Link</button>

<!-- 次要样式 -->
<button type="button" class="v-button secondary">Button</button>

<!-- 中间色样式 -->
<button type="button" class="v-button demitint">Button</button>

<!-- 固定宽样式 -->
<button type="button" class="v-button primary min-width">min-width</button>

<!-- 图标样式 -->
<button type="button" class="v-button primary">
    <i class="icon-cloud-upload"></i>
</button>

获得不同尺寸的按钮

添加 .large .small .mini 类改变按钮尺寸 
<!-- 按钮尺寸 -->
<button type="button" class="v-button primary large">Button</button>
<button type="button" class="v-button primary">Button</button>
<button type="button" class="v-button primary small">Button</button>
<button type="button" class="v-button primary mini">Button</button>

禁用状态

按钮禁用 .disabled 
<!-- 禁用状态 -->
<button type="button" class="v-button primary disabled">Button</button>

按钮组,将多个按钮组合在一起

按钮组支持预置颜色.primary .secondary .demitint和尺寸.large .small .mini
可定义单一按钮颜色，不建议单独定义尺寸
Left Middle Right 

<!-- 预置样式 -->
<div class="v-button-group">
    <a href="#" class="v-button primary">Left</a>
    <a href="#" class="v-button primary">Middle</a>
    <a href="#" class="v-button primary">Right</a>
</div>

<!-- 自定义样式 -->
<div class="v-button-group">
    <a href="#" class="v-button"><i class="icon-align-left"></i></a>
    <a href="#" class="v-button"><i class="icon-align-center"></i></a>
    <a href="#" class="v-button"><i class="icon-align-right"></i></a>
    <a href="#" class="v-button demitint"><i class="icon-layers"></i></a>
</div>

<!-- 图标样式 -->
<div class="v-button-group">
    <a href="#" class="v-button"><i class="icon-skip-back"></i></a>
    <a href="#" class="v-button"><i class="icon-stop"></i></a>
    <a href="#" class="v-button"><i class="icon-skip-forward"></i></a>
</div>

标签按钮

标签按钮支持预置颜色.primary .secondary .demitint和尺寸.large .small .mini
Like 2,048 
<!-- 预置样式 -->
<div class="v-button-label primary">
    <a href="#" class="v-button"><i class="icon-heart"></i> Like </a>
    <span class="label">
        2,048
    </span>
</div>

图标按钮

图标按钮同样支持内置样式
Send 
<!-- 预置样式 -->
<div class="v-button-label">
    <a href="#" class="v-button"><i class="icon-heart"></i> Like </a>
    <span class="label">
        2,048
    </span>
</div>

块级按钮

添加 .block 类可以将其拉伸至父元素100%的宽度，按钮变为块级元素
Button Button 
<!-- 预置样式 -->
<a href="javascript:;" class="v-button primary block">Button</a>

<!-- 图标样式 -->
<a href="javascript:;" class="v-button primary label-icon block">
    <i class="icon-mail"></i>
    Button
</a>
